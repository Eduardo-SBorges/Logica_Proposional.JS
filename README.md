Verificador de Regras de Inferência

Programa que verifica se existem deduções validas nas proposições inseridas pelo usuário e, se sim, exibe a qual o teorema está inserida.

Exemplo:

1ª hipótese: p -> ~r  |
2ª hipótese: ~r -> q
------------------------------
Dedução: p -> q
Teorema: Silogismo Hipotético
